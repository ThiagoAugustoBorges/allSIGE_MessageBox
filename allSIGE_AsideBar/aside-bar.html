allSIGE Message Box - JQuery

allSIGE Message Box é um plugin JQuery simples para notificações de mensagens em uma página da Web em um determinado momento ou em um clique de botão (ou mesmo uma parte de uma função do botão). Decida por si mesmo quando você irá disparar uma notificação. Use apenas como um notificador de mensagens, evite o uso de qualquer tipo de input de dados pois o propósito inicial do plugin não é esse.

Instalação

Requerido JQuery 1.1x~ (2.x~ Recomendado).

Utilização de classes do Twitter Bootstrap para os botões, porém sem o uso obrigatório.

<link rel="stylesheet" href="css/allSIGE-dynamic-messageBox.css"/>
<script src="js/jquery.js"></script/>
<script src="js/allSIGE-dynamic-messageBox.js"></script/>
